# Fieldnotes

🏕️ Notes from the field — exploratory data analysis. 

## About

This repository contains a collection of Jupyter notebooks used to explore public datasets and practice data analysis workflows.

The focus is on:

* exploratory data analysis (EDA)
* pandas-based transformations
* basic visualization (matplotlib, seaborn)
* working with time-series and categorical data

These notebooks are not intended to be production-ready analyses. They are iterative, lightweight, and focused on learning and exploration.

## Tools

* Python
* pandas
* matplotlib
* seaborn
* Maybe Julia will make an appearance
